# API documentation

## End-Points

### GET /data/dev/id
  Description: Get data by ID

### POST /data/dev
  Description: Create new data in file storage

### PATCH /data/dev/id
  Description: Update existing data by ID

### DELETE /data/dev/id
  Description: Delete data by ID
